# NovaTicket AI Recommender & Sentiment Analyzer

> **Status**: 🚧 In Development

## Overview

NovaTicket is an event platform with integrated AI features:
- **Personalized Event Recommendations** — Content-based, Collaborative Filtering, and Hybrid engine.
- **Sentiment Analysis** — ML-based classification of user reviews into Positive / Neutral / Negative.

## Tech Stack

| Layer | Technology |
|---|---|
| Backend | Python 3.11, FastAPI, SQLAlchemy 2.0, Alembic |
| Database | Microsoft SQL Server 2022 |
| ML | Scikit-learn, Pandas, NumPy, joblib |
| Frontend | React 18, Vite |
| Infrastructure | Docker, Docker Compose, GitHub Actions |
